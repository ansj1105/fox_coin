# 채굴·레벨·EXP 스펙

채굴 시작 조건, 영상 연동, 레벨별 제한, EXP 시스템 정리입니다.

---

## 1. 채굴 기본 조건

### 채굴 시작 필수 조건
- **이메일 인증 완료**
- **부스터 영상 1회 시청**

### 영상 1회 효과
| 항목 | 내용 |
|------|------|
| 채굴 가능 시간 | 1시간 (영상 1회 = 1시간 연료) |
| 연속 시청 보너스 | 없음 |
| 다음 영상 시청 | 반드시 1시간 종료 후에만 가능 |
| 영상 미시청 시 | 채굴 불가 |
| 구조 | **영상 = 채굴 연료** |

---

## 2. 레벨 시스템 (총 9레벨)

- 레벨이 오를수록 **하루 시청 가능 영상 수** 증가, **하루 최대 채굴량** 증가
- LV9: 하루 MAX 10 KORI, 영상 24개 필요
- 하루 최대 채굴량은 레벨별로 고정

### 레벨별 하루 제한표

| 레벨 | 영상 시청 | 하루 최대 채굴량 | 시간당 채굴량 | 분당 채굴량 |
|------|----------|------------------|---------------|-------------|
| LV1 | 5개 | 1 KORI | 0.20 | 0.00333 |
| LV2 | 7개 | 2 KORI | 0.286 | 0.00477 |
| LV3 | 9개 | 3.2 KORI | 0.356 | 0.00593 |
| LV4 | 12개 | 4.8 KORI | 0.40 | 0.00667 |
| LV5 | 15개 | 6.5 KORI | 0.433 | 0.00722 |
| LV6 | 18개 | 8.0 KORI | 0.444 | 0.00740 |
| LV7 | 20개 | 9 KORI | 0.45 | 0.00750 |
| LV8 | 22개 | 9.6 KORI | 0.436 | 0.00727 |
| LV9 | 24개 | 10 KORI | 0.417 | 0.00695 |

- **영상 1회당 채굴량** = `하루 최대 채굴량 / 영상 시청 수` (예: LV1 = 1/5 = 0.2 KORI)

## 3. 경험치(EXP) 시스템 (채굴 연동)

### EXP 규칙
| 항목 | 내용 |
|------|------|
| 1 KORI | 1 EXP |
| 영상만 보고 채굴 안 하면 | EXP 0 |
| 하루 최대 EXP | 하루 최대 채굴량과 동일 |

### 레벨업 누적 필요 EXP
| 구간 | 누적 필요 EXP |
|------|---------------|
| LV1 → LV2 | 5 |
| LV2 → LV3 | 15 |
| LV3 → LV4 | 35 |
| LV4 → LV5 | 70 |
| LV5 → LV6 | 130 |
| LV6 → LV7 | 220 |
| LV7 → LV8 | 350 |
| LV8 → LV9 | 520 |

- 위 값은 **누적** EXP (LV1 시작 시 0, LV2 되려면 5, LV3 되려면 15, …)

## 4. 친구 초대 & 래퍼럴 (REFERRAL_AND_INVITE_SPEC 연동)

- 친구 초대 = 채굴 + 수익 + 성장
- **채굴 속도 보너스**: 일일 최대 채굴량 유지, 영상 1회당 채굴량만 +3%~+22% (초대 인원 구간별)
- **래퍼럴 수익**: 하부 채굴 KORI 기준 3%~13%
- **안전장치**: 이메일 인증 필수, 영상 시청+채굴 기록 필수, 동일 IP/기기 중복 초대 무효
- **친구 초대 EXP**: 초대 1명당 1 EXP
